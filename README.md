# 🌴 SmartNusa

> **Discover Authentic Indonesia — One Smart Itinerary at a Time.**

SmartNusa is more than just an all-in-one travel app; it is a pioneering **Social Impact Technology** ecosystem that elevates the standard travel experience with AI-driven smart itineraries. We empower local communities, ensure traveler safety, and promote sustainable tourism across Indonesia.

🌐 **Live Demo:** [Visit SmartNusa Website](https://nadhif-royal.github.io/smartnusa/) *(Update link ini jika ada)*

---

## 🚀 Key Features

### 🤖 Nusa AI & Smart Itinerary
A smart chatbot and itinerary builder powered by AI Tourism Demand Prediction to forecast crowds, provide capacity warnings, and generate personalized travel plans in seconds.

### 🛡️ Nusa Guard
Integrated with the BMKG API for real-time weather predictions, disaster alerts, GPS tracking, and safe routing. Travel with ultimate peace of mind.

### 🤝 Nusa Buddy & NusaGo
* **Nusa Buddy:** A local service marketplace to hire verified local photographers, certified guides, and transport providers directly.
* **NusaGo:** Book verified tour packages with transparent pricing and escrow protection.

### 🌿 NusaGreen & NusaPoints
* **NusaGreen:** Travel sustainably with carbon footprint tracking and eco-friendly route recommendations.
* **NusaPoints:** A gamification system that rewards users for bookings and making sustainable travel choices.

### 👗 Nusa Outfit Match
AI recommends the perfect outfit tailored to your destination's vibe and background for aesthetic photos.

## 🛠️ Tech Stack

This landing page and interactive prototype are built with:
* **HTML5** - Semantic and accessible markup.
* **CSS3** - Custom styling with a modern, responsive layout (Flexbox & CSS Grid).
* **Vanilla JavaScript** - For dynamic interactions, mobile menu, and logic handling (e.g., Smart Itinerary generator).
* **AOS (Animate On Scroll)** - For smooth, scrolling animations.
* **Phosphor Icons** - For clean, consistent iconography.

## 💻 Getting Started

To run this project locally on your machine:

1. Clone the repository:
   ```bash
   git clone [https://github.com/nadhif-royal/smartnusa.git](https://github.com/nadhif-royal/smartnusa.git)
   ```

2.  Navigate into the project directory:
    ```bash
    cd smartnusa
    ```
3.  Open `index.html` in your preferred web browser. No local server is strictly required, but using an extension like VS Code's "Live Server" is recommended for the best experience.
````
